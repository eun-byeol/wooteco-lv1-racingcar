### 구현할 기능 목록 🚗

- [x] 자동차 이름을 입력받는다.
    - 쉼표를 기준으로 구분한다.
- [x] 시도할 회수를 입력받는다.
- [x] 0 ~ 9 사이 랜덤값을 구한다.
- [x] 랜덤 값이 4 이상일 경우 전진한다.
- [x] 실행 결과를 출력한다.
    - 출력 형식은 `이름 : -`
- [x] 우승자를 발표한다.
    - 모든 참여자가 0칸 이동했다면 모두 우승이다.

### 예외 상황

- 예외 발생시 다시 입력받는다.

- 자동차 이름 입력 받을 때
    - [x] 5자를 초과하면 예외
    - [x] 영어나 한글이 아니면 예외
    - [x] 참가자가 1명 이하일 경우 예외
    - [x] 자동차 이름에 중복이 있을 경우 예외

- 시도할 회수를 입력할 때
    - [x] 음수면 예외
    - [x] 100을 초과하면 예외
